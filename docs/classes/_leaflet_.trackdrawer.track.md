# Class: Track

Main entry point.

*__emits__*: TrackDrawer:start Fired when an edit has started

*__emits__*: TrackDrawer:done Fired when all pending edits are done

*__emits__*: TrackDrawer:failed Fired if an error occured

**↳ Track**
